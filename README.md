# 🎮 Tic-Tac-Toe Game

A modern Tic-Tac-Toe game built with **React**. The application allows two players to compete, edit their names, keeps track of every move, detects winners or draws, and lets players restart the game.

---

## ✨ Features

- 🎯 Two-player gameplay
- ✏️ Edit player names
- 🔄 Automatic player turn switching
- ✅ Winner detection
- 🤝 Draw detection
- 📜 Move history (Game Log)
- 🔒 Prevents selecting an occupied square
- 🔄 Restart game anytime
- ⚛️ Built with React Hooks

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ibrahimyasser450/Tic-Tac-Toe-Game.git
```

### Navigate to the project

```bash
cd tic-tac-toe
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 🎮 How to Play

1. Player **X** starts first.
2. Players take turns selecting an empty square.
3. The first player to align **three symbols** horizontally, vertically, or diagonally wins.
4. If all squares are filled without a winner, the game ends in a draw.
5. Click **Restart** to begin a new game.
